## Org Design Principles

1. **Flat hierarchy** - We're too small for bureaucracy. Alex makes final calls.
2. **Agent autonomy** - Agents operate within defined scope without micromanagement.
3. **Skill reuse** - Skills provide shared capabilities; agents don't duplicate functionality.
4. **Context sharing** - Business context lives centrally; all agents can access it.
5. **Project-based work** - PSO tracks initiatives; work is organized as projects.

## Future Roles (As We Grow)

When funding allows, we'll need:
- **Engineering Manager** - Own tech execution, coordinate agent workforce
- **DevOps/SRE** - Infrastructure, reliability, deployment
- **QA/Testing** - Agent validation, testing frameworks
- **Security Officer** - Compliance, access control, data protection

Until then: Alex wears all hats, with agent assistance.
